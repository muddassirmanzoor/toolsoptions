# toolsoptions

A comprehensive PDF and document processing tool suite built with Laravel (PHP) and Blade templates.

## Features

- PDF compression
- PDF splitting and merging
- PDF to Word/Excel conversion
- Word/Excel to PDF conversion
- PDF watermarking
- PDF protection and unlocking
- PDF cropping and redaction
- And many more document processing tools

## Tech Stack

- **Backend**: PHP (Laravel)
- **Frontend**: Blade templates
- **Database**: MySQL
- **Additional**: Node.js for some processing tasks

## Installation

1. Clone the repository
2. Install PHP dependencies: `composer install`
3. Install Node.js dependencies: `npm install`
4. Copy `.env.example` to `.env` and configure
5. Set up your database and update `.env`
6. Run migrations if applicable

## Requirements

- PHP 7.4+
- MySQL
- Node.js
- Composer
