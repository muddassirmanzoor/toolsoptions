About PDF Editing

Edit your PDF documents with our powerful editor. You can:

  • Select text in the PDF to edit it
  • Add new text, shapes, and images
  • Draw annotations and highlights
  • Edit multiple pages

Tip: After uploading, select text on the PDF to edit it. Use the toolbar to add new elements.

PDF Preview

Editing Tools

  • Text: Click any text in PDF to edit, or use Text button to add new text
  • Image: Insert images into the PDF
  • Shapes: Draw rectangles, circles, and lines
  • Zoom: Adjust the view size

Tip: Click any text in the PDF preview to edit it directly. All pages are scrollable. New text/shapes added via toolbar will appear on the current page.
